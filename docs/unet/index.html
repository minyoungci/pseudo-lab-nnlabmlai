#

U-Net

다음은 U-Net: Convolutional Networks for Biomedical Image Segmentation 논문을 코드로 구현한 것입니다.

U-Net은 contracting path와 expansive path로 구성되어 있습니다. contracting path는 컨볼루션 레이어와 풀링 레이어들이 이어져있는 구조이며, 특성 맵의 해상도가 점진적으로 줄어듭니다. expansive path는 업샘플링 레이어와 컨볼루션 레이어들이 이어져있는 구조로 특성 맵의 해상도가 점진적으로 증가합니다.

expansive path의 각 step에서는, 대응하는 contracting path에서의 특성 맵과 expansive path의 특성 맵을 이어붙입니다.

U-Net을 Carvana 데이터셋으로 훈련하는 코드 입니다.
